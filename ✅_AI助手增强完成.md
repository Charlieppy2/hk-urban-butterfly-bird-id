## 🔧 使用方法

### 后端（已自动集成）
增强功能已自动启用，无需额外配置。

### 前端（可选增强）
可以在发送请求时添加 `user_id` 来启用个人化：

```javascript
const response = await axios.post(`${API_URL}/api/chat`, {
  message: userMessage.text,
  context: {
    lastPrediction: prediction,
    historyCount: history.length
  },
  user_id: 'user_123' // 可选：启用个人化功能
});
```
