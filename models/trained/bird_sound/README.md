# Bird Sound Model

This directory contains the trained bird sound identification model.

## Model Files

Place your trained model file(s) here. Common formats:
- `.h5` (Keras/TensorFlow)
- `.pkl` (Pickle)
- `.pt` or `.pth` (PyTorch)
- `.pb` (TensorFlow SavedModel)
- `.onnx` (ONNX format)

## Usage

Run the model using:
```bash
python run_bird_sound_model.py
```

Or from the project root:
```bash
python models/trained/bird_sound/run_bird_sound_model.py
```
