# 如何查看訓練進度

## 方法1：使用檢查腳本（快速查看）
```powershell
cd models/training
python check_training.py
```

## 方法2：查看模型文件更新時間
```powershell
Get-Item "models\trained\model.h5" | Select-Object Name, Length, LastWriteTime
```
如果 `LastWriteTime` 持續更新，說明訓練正在進行。

## 方法3：查看實時訓練輸出（最詳細）
1. 打開新的 PowerShell 窗口
2. 執行：
```powershell
cd "d:\School_IVE\IVE-HD IT114118\04_Sem04\ITP4221_IT_ICT_1( Artificial Intelligence Technologies ) by TUNG CHI CHEUNG\EA_project\HK Urban Ecological Portable Identification\models\training"
python train_model.py
```

這樣可以看到：
- 每個 epoch 的進度條
- 訓練準確率 (accuracy)
- 驗證準確率 (val_accuracy)
- Loss 值
- Top-3 準確率

## 方法4：檢查進程是否運行
```powershell
Get-Process python
```
如果有輸出，說明訓練正在運行。

## 訓練完成標誌
- 看到 "Training Results:" 輸出

## 訓練時間估算
- 每個 epoch 約 3-5 分鐘
- 100 epochs 約需 5-8 小時
- 如果啟用 Early Stopping，可能會提前結束
